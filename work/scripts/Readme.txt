This file will guide you through the steps of installation for MOOS Project Viewer software. The installation is of unzip and run
type.

Here is what you have to do:

1. Unzip to the folder you want to install the application. You already did that if you are reading this file.
2. Just run the scripts.
	a. Run MOOS.bat for Windows operating systems.
	b. Run MOOS.sh for Linux, Unix and Mac operating systems.
